# Car Dealership Review Application

## Repository Name
car-dealership-project

## Project Overview
This project is a full-stack Car Dealership Review Application developed as part of the Full Stack Software Developer Capstone Project.

The application allows users to:
- View car dealership branches
- Register and log in securely
- View dealer details and customer reviews
- Post reviews for dealerships
- Analyze review sentiments using AI-based sentiment analysis

## Technologies Used
- React
- Django
- Node.js
- Flask
- MongoDB
- SQLite3
- Docker
- Kubernetes
- GitHub Actions
- IBM Cloud Code Engine

## Features
- User Authentication
- Dealer Listing by State
- Dealer Details and Reviews
- Sentiment Analysis Integration
- Responsive Frontend Design
- CI/CD Pipeline with GitHub Actions
- Cloud Deployment Support
